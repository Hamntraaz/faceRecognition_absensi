# 📸 Face Absensi

Aplikasi **Face Absensi** adalah sistem presensi modern berbasis **pengenalan wajah (face recognition)** yang dibangun menggunakan Flutter.
Aplikasi ini membantu proses absensi menjadi **lebih cepat, akurat, dan tanpa kontak**.

---

## ✨ Fitur Utama

* 👤 **Face Recognition**
  Absensi menggunakan wajah tanpa perlu input manual

* ⏱️ **Realtime Attendance**
  Pencatatan kehadiran secara langsung

* 📱 **Cross Platform**
  Berjalan di Android, iOS, dan Web (Flutter)

* 🔐 **Aman & Praktis**
  Mengurangi kecurangan seperti titip absen

* ☁️ **Integrasi Backend (Firebase/API)**
  Data tersimpan secara online (opsional)

---

## 🛠️ Teknologi yang Digunakan

* 💙 Flutter (UI Framework)
* 🎯 Dart (Programming Language)
* 🔥 Firebase (Authentication / Database - opsional)
* 📷 Camera & Face Detection

---

## 🚀 Cara Menjalankan Project

### 1. Clone Repository

```bash
git clone https://github.com/USERNAME/face_absensi.git
cd face_absensi
```

### 2. Install Dependencies

```bash
flutter pub get
```

### 3. Jalankan Aplikasi

```bash
flutter run
```

---

## ⚙️ Persyaratan

Pastikan sudah terinstall:

* Flutter SDK
* Android Studio / VS Code
* Emulator atau perangkat fisik

Cek dengan:

```bash
flutter doctor
```
